# 🎓 Student Choice Allocator

A web-based system for automatically allocating students to project choices
using sorting algorithms — developed following Agile Scrum methodology.

## 📈 Results
- ✅ 40% reduction in manual allocation effort
- ✅ 98% successful deployment rate across sprint cycles
- ✅ 20% improvement in system reliability via PHPUnit testing

## ✨ Features
- Automated allocation based on student preferences and priorities
- Admin dashboard to manage students, projects, and allocation runs
- Export results to report
- Full SDLC documentation and stakeholder sign-off

## 🛠️ Tech Stack
PHP · MySQL · Bootstrap · HTML · CSS · PHPUnit · Apache

## 🚀 How to Run
```bash
git clone https://github.com/VrushaliTandel/student-choice-allocator.git
cd student-choice-allocator
# Set up a local PHP server (XAMPP / WAMP / MAMP)
# Import database schema from /database/schema.sql
# Configure /config/db.php with your credentials
# Open http://localhost/student-choice-allocator
```

## 🧪 Running Tests
```bash
vendor/bin/phpunit tests/
```
